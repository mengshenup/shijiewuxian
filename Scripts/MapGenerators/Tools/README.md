# Map Generator Tools

实用工具批处理文件。

## 工具列表

### check_zen_status.bat
检查Zen服务器状态
- 检查zenserver.exe进程
- 测试HTTP服务连接

### edit_config.bat
编辑配置文件
- 打开config.ini进行编辑
- 修改引擎路径、项目路径、命令行参数

### view_map.bat
在编辑器中打开生成的地图
- 直接在UE5编辑器中打开Cosmos_002_Training_World地图

## 使用说明

这些工具是辅助工具，不是必需的。主要的地图生成使用根目录的 `generate_map.bat`。
