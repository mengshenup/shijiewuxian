# 在编辑器中执行Python脚本的说明

## 问题分析

命令行执行 `-ExecutePythonScript` 参数似乎不起作用。引擎启动了，但没有执行我们的Python脚本。

## 解决方案：在编辑器中执行

1. 打开UE5编辑器
2. 打开 Window → Developer Tools → Output Log
3. 切换到 Python 标签
4. 执行以下代码：

```python
import sys
sys.path.append('D:/001xm/shijiewuxian/Scripts/MapGenerators')
import generate_cosmos_002_training_world
generate_cosmos_002_training_world.main()
```

## 为什么命令行不工作？

可能的原因：
1. `-ExecutePythonScript` 参数在 UE5.7.0 中可能需要不同的格式
2. 可能需要使用 `-run=pythonscript` 或其他参数
3. 命令行模式可能不支持Python脚本执行
