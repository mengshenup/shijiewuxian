# UE5 坐标系说明

## 坐标轴定义

**UE5 使用左手坐标系：**
- **X 轴（红色）**：前/后（Forward/Backward）
  - +X = 前方（Forward）
  - -X = 后方（Backward）

- **Y 轴（绿色）**：右/左（Right/Left）
  - +Y = 右侧（Right）
  - -Y = 左侧（Left）

- **Z 轴（蓝色）**：上/下（Up/Down）
  - +Z = 上方（Up）
  - -Z = 下方（Down）

## 旋转（Rotator）

**Rotator(Pitch, Yaw, Roll)：**
- **Pitch**：俯仰角（绕 Y 轴旋转）
  - +Pitch = 向上看
  - -Pitch = 向下看

- **Yaw**：偏航角（绕 Z 轴旋转）
  - +Yaw = 向右转
  - -Yaw = 向左转

- **Roll**：翻滚角（绕 X 轴旋转）
  - +Roll = 向右倾斜
  - -Roll = 向左倾斜

## 训练室布局（俯视图）

```
                    +X (前方)
                       ↑
                       |
        ┌──────────────┼──────────────┐
        │              |              │
        │    左房间    |   中间房间    │   右房间
        │              |              │
-Y ←────┼──────────────┼──────────────┼────→ +Y
(左侧)  │              |              │   (右侧)
        │              |              │
        └──────────────┼──────────────┘
                       |
                       ↓
                    -X (后方)
```

## 正确的墙壁位置

### 地板和天花板
- **地板**：Z = 0
- **天花板**：Z = 400

### 前后墙（垂直于 X 轴）
- **前墙（FrontWall）**：
  - 位置：X = +900（前方）
  - 旋转：Yaw = 0°（面向 -X）
  
- **后墙（BackWall）**：
  - 位置：X = -900（后方）
  - 旋转：Yaw = 180°（面向 +X）

### 左右墙（垂直于 Y 轴）
- **左墙（LeftWall）**：
  - 位置：Y = -400（左侧）
  - 旋转：Yaw = -90°（面向 +Y）
  
- **右墙（RightWall）**：
  - 位置：Y = +400（右侧）
  - 旋转：Yaw = 90°（面向 -Y）

### 隔断（垂直于 Y 轴）
- **左隔断（LeftPartition）**：
  - 位置：Y = -300
  - 旋转：Yaw = -90°
  
- **右隔断（RightPartition）**：
  - 位置：Y = +300
  - 旋转：Yaw = 90°

## 房间尺寸

**总尺寸：**
- X 轴（前后）：1800 单位（-900 到 +900）
- Y 轴（左右）：800 单位（-400 到 +400）
- Z 轴（高度）：400 单位（0 到 400）

**每个房间：**
- X 轴：1800 单位
- Y 轴：约 267 单位（800 / 3）
- Z 轴：400 单位

## 常见错误

❌ **错误示例（当前代码）：**
```python
# 前墙使用 Y 轴 - 错误！
unreal.Vector(0, 400, 200)  # 这实际上是右墙的位置

# 左墙使用 X 轴 - 错误！
unreal.Vector(-900, 0, 200)  # 这实际上是后墙的位置
```

✅ **正确示例：**
```python
# 前墙使用 X 轴
unreal.Vector(900, 0, 200)  # 前方

# 左墙使用 Y 轴
unreal.Vector(0, -400, 200)  # 左侧
```

## PlayerStart 位置

**推荐位置：**
- 位置：`unreal.Vector(-800, 0, 100)`（后方中央，离地 100）
- 旋转：`unreal.Rotator(0, 0, 0)`（面向前方 +X）

这样玩家会：
- 站在后方
- 面向前方
- 看到整个训练室

## 光源位置

**点光源（每个房间一个）：**
- 左房间：`unreal.Vector(0, -200, 350)`
- 中间房间：`unreal.Vector(0, 0, 350)`
- 右房间：`unreal.Vector(0, 200, 350)`

**方向光（太阳）：**
- 位置：`unreal.Vector(0, 0, 1000)`（高空）
- 旋转：`unreal.Rotator(-45, 45, 0)`（45度角照射）
