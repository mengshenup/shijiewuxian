# 地图生成器执行说明

## 快速执行

1. **双击运行：** `generate_map.bat`

2. **等待编辑器打开**

3. **在编辑器中执行Python代码：**
   - 打开：Window → Developer Tools → Output Log
   - 切换到 "Python" 标签
   - 复制粘贴以下代码：
   
   ```python
   import sys
   sys.path.append('D:/001xm/shijiewuxian/Scripts/MapGenerators')
   import generate_cosmos_002_training_world
   generate_cosmos_002_training_world.main()
   ```
   
   - 按 Enter 执行

4. **等待完成**
   - 看到 "SUCCESS!" 消息表示成功
   - 地图位置：`Content/Maps/Cosmos_002_Training_World.umap`

## 为什么不能命令行执行？

UE5.7.0的 `-ExecutePythonScript` 参数不工作。引擎会启动但忽略Python脚本。

## 生成的内容

- **TrainingRoom_Main** - 3个相连的训练房间
- **PlayerStart_Center** - 玩家出生点
- **DirectionalLight_Sun** - 主光源
- **PointLight_Left/Center/Right** - 房间照明
- **GameMode** - FPSTrainingGameMode

## 验证

在Content Browser中打开地图，点击Play按钮测试。
